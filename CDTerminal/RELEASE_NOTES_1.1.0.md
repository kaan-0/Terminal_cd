# CD Terminal 1.1.0

Versión instalable con una interfaz modular y el nuevo Configurador de equipos IoT.

## Novedades

- Inicio vacío: los módulos ocupan el área de trabajo cuando se seleccionan.
- Configurador IoT con conexión serial independiente.
- Selector de puerto COM, baud rate, DTR y RTS.
- Terminal RX/TX completa dentro del configurador.
- Envío individual de parámetros del firmware.
- Configuración DHCP o IP estática.
- IP y máscara del servidor.
- Puerto HTTP, período y URL del conector.
- Lectura completa de la configuración almacenada en EEPROM.
- Comando de reinicio `LC0001CresetZ` con confirmación.
- Diseño compacto adaptable para pantallas de laptop.
- Se conservan terminal serial, Modbus RTU, polling, CSV/TXT, gráfica, perfiles y herramientas existentes.

## Requisitos

- Windows 10 u 11 de 64 bits.
- Microsoft Edge WebView2 Runtime.
- Puerto COM o adaptador serial compatible.
